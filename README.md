# Vida & Carteira

Sistema pessoal privado de trajetória de vida, organizado em seis pilares (Saúde, Família, Relacionamentos, Trabalho, Material, Espiritual), do nascimento até cinco anos à frente.

Privado por design. Todos os dados são criptografados com AES-GCM no navegador.
